## 🎯 Common Tasks

### Task: "I need to style a heading"
1. Open [DESIGN_SYSTEM_QUICK_REF.md](./DESIGN_SYSTEM_QUICK_REF.md#typography-quick-reference)
2. Copy the appropriate class: `heading-1`, `heading-2`, etc.
3. Apply to your element: `<h1 className="heading-1">Title</h1>`

### Task: "I need to create a button"
1. Open [DESIGN_SYSTEM_QUICK_REF.md](./DESIGN_SYSTEM_QUICK_REF.md#buttons)
2. Choose variant: primary, secondary, outline, or ghost
3. Copy example: `<button className="btn btn-primary">Click Me</button>`

### Task: "I need to see all colors"
1. Open [color-palette.html](./color-palette.html) in browser
2. Browse color swatches
3. Copy hex codes or CSS variables

### Task: "I need to create a card"
1. Open [DESIGN_SYSTEM_QUICK_REF.md](./DESIGN_SYSTEM_QUICK_REF.md#cards)
2. Copy card structure
3. Customize content

### Task: "I need spacing between elements"
1. See [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md#spacing-system)
2. Use classes: `mt-4`, `mb-6`, `p-8`, etc.
3. Follow 4px grid system

## 🆘 Troubleshooting

### "Styles not working"
1. Check if `src/styles/design-system.css` exists
2. Verify `src/index.css` imports it
3. Clear browser cache
4. Restart dev server

### "Colors look wrong"
1. Open [color-palette.html](./color-palette.html) to verify
2. Check if using correct CSS variables
3. Verify Tailwind config is updated

### "Need help with spacing"
1. See [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md#spacing-system)
2. Use 4px grid: `space-1` (4px), `space-2` (8px), etc.
3. View [design-system-example.html](./design-system-example.html) for examples
