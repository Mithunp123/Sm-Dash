# Images Directory

Place your logo and coordinator images in this directory.

## Required Images

### Logos (Place in `/public/images/`)

1. **ksrct-logo.png** - K.S.Rangasamy College of Technology logo
   - Recommended size: 200x200px or higher
   - Format: PNG with transparent background

2. **smv-logo.png** - SM Volunteers logo
   - Recommended size: 300x300px or higher
   - Format: PNG with transparent background

3. **NGO Logos:**
   - **ngo-atchaym.png** - Atchaym Trust logo
   - **ngo-bhumi.png** - Bhumi logo
   - **ngo-tqi.png** - Talent Quest for India (TQI) logo
   - Recommended size: 150x150px each
   - Format: PNG with transparent background

### Coordinator Photos (Place in `/public/images/coordinators/`)

1. **dr-gopal.jpg** - Dr. Gopal (Patron)
   - Recommended size: 200x200px
   - Format: JPG or PNG
   - Square aspect ratio

2. **mr-palaniappan.jpg** - Mr. Palaniappan (Coordinator)
   - Recommended size: 200x200px
   - Format: JPG or PNG
   - Square aspect ratio

3. **Forum Leaders:**
   - **gopalakrishnan.jpg** - GopalaKrishnan
   - **mythili-gnanmangai.jpg** - Mythili Gnanmangai
   - **raj-kumar.jpg** - Raj Kumar
   - Recommended size: 150x150px each
   - Format: JPG or PNG
   - Square aspect ratio

## Notes

- If images are not found, the application will automatically fall back to text placeholders
- Images should be optimized for web (compressed but good quality)
- Use square images for coordinator photos for best display
- PNG format recommended for logos to support transparency
